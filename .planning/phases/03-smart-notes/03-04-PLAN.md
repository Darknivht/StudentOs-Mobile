- Export type SummaryLength = 'short' | 'medium' | 'long'
- Export function generateNoteSummary(noteContent: string, length: SummaryLength, onChunk: (chunk: string) => void, oncomplete: (fullText: string) => void, onError: (error: Error) => void): AbortController
  - Build system message: "You are a helpful study assistant. Generate a {length} summary of the following note. A short summary is 1-2 sentences. A medium summary is 1 paragraph. A long summary is 2-3 paragraphs covering all key points."
  - Build user message: noteContent (strip HTML tags first: content.replace(/<[^>]*>/g, ''))
  - Create AbortController for cancellation
  - Get AI provider: AIProviderFactory.getInstance()

- Export function generateSocraticQuestion(noteContent: string, onChunk: (chunk: string) => void, onComplete: (fullText: string) => void, onError: (error: Error) => void): AbortController
  - System message: "You are a Socratic tutor. Given the student's note, ask a thought-provoking question that helps them think deeper about the material. Do NOT give the answer. Ask only one question at a time. Be encouraging and curious."
  - User message: "Here is my note:\n\n" + strippedContent
  - Use streamChat with the same pattern as aiSummary
  - Return AbortController
- Export function generateSocraticFollowup(noteContent: string, chatHistory: array of {role: string, content: string}, onChunk, onComplete, onError): AbortController
  - System message: "You are a Socratic tutor continuing a conversation about the student's note. Ask follow-up questions that guide them to deeper understanding. Do NOT give direct answers. Build on their responses."
  - Messages array: system message + chat history (user and assistant messages from the Socratic conversation)
  - Use streamChat
  - Return AbortController

- Import useSubscription, useNotes, SUBSCRIPTION_TIERS from constants
- Import generateNoteSummary, generateSocraticQuestion, generateSocraticFollowup from services
- Track AI calls today: state variable aiCallsToday, loaded from Supabase query on mount
  - Query: supabase.from('ai_usage').select('id', {count: 'exact', head: true}).eq('user_id', userId).eq('feature', 'notes').gte('created_at', startOfTodayUTC)
  - Alternative: use a simpler localStorage counter with midnight UTC reset
- Export function useNoteAI()
- State: isGenerating: boolean, streamingText: string, error: string | null, abortController: AbortController | null
- Computed: aiCallsLimit from SUBSCRIPTION_TIERS[tier].aiCalls, isQuotaExceeded = aiCallsToday >= aiCallsLimit
- Method generateSummary(noteId: string, noteContent: string, length: SummaryLength): void
  - Check quota: if isQuotaExceeded, show Alert "Daily AI limit reached" with tier info, return
  - Set isGenerating=true, streamingText='', error=null
  - Create abortController
  - Call generateNoteSummary(noteContent, length, onChunk, onComplete, onError)
  - onChunk: append chunk to streamingText state
  - onComplete: set isGenerating=false, save summary to note via updateNote(noteId, {summary: fullText}), increment aiCallsToday
  - onError: set isGenerating=false, set error message
- Method startSocratic(noteContent: string): AbortController
  - Check quota before first question
  - Set isGenerating=true, streamingText=''
  - Call generateSocraticQuestion(noteContent, onChunk, onComplete, onError)
  - Return abortController for caller to cancel if needed
- Method continueSocratic(noteContent: string, chatHistory: array): AbortController
  - Same pattern, calls generateSocraticFollowup
  - Increment aiCallsToday on each AI call
- Method cancel(): void - abort current generation
- Method reset(): void - clear streaming state
- Return all state and methods
