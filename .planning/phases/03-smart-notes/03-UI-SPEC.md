# Phase 3 — UI Design Contract: Smart Notes

> Visual and interaction contract for the Smart Notes feature. Generated by gsd-ui-researcher, verified by gsd-ui-checker. Covers all 4 sub-plans (03-01 through 03-04). React Native + Expo — no shadcn (web-only).

## Spacing Scale

Declared values (all multiples of 4, sourced from `src/lib/theme.ts`):

| Token | Value | Usage in Smart Notes                                                           |
| ----- | ----- | ------------------------------------------------------------------------------ |
| xs    | 4px   | Icon gaps, inline padding, progress bar height, active dot                     |
| sm    | 8px   | Compact element spacing, search bar clear button padding, pill gap             |
| md    | 16px  | Card padding, search bar padding, section item spacing, icon container padding |
| lg    | 24px  | Section padding, action row padding, screen horizontal padding                 |
| xl    | 32px  | Layout gaps, FAB offset from edge, bottom sheet content padding                |
| 2xl   | 48px  | Major section breaks, scroll content bottom padding                            |
| 3xl   | 64px  | Page-level spacing (unused in this phase)                                      |

Exceptions:

- **FAB button size**: 56×56px (14×14 multiples of 4 — standard Material FAB, not on 8-point grid)
- **FAB hit slop**: 44×44px for CameraOCRButton and AttachFileButton (touch target accessibility, per Plan 02)
- **Bottom tab bar height**: 60 + insets.bottom (per MainNavigator existing pattern)
- **Swipe action button width**: 80px (fixed, not grid-aligned — standard for swipe actions)
- **Source accent bar**: 4px width (matches xs token)
- **Progress bar height**: 4px (matches xs token)

## Typography

4 declared roles (sourced from `src/lib/theme.ts` typography values):

| Role    | Size                     | Weight         | Line Height | Usage in Smart Notes                                                                            |
| ------- | ------------------------ | -------------- | ----------- | ----------------------------------------------------------------------------------------------- |
| Body    | 16px (typography.base)   | 400 (regular)  | 1.5 (24px)  | Note preview text, chat messages, descriptions, auto-save status, quota text                    |
| Label   | 14px (typography.sm)     | 400 (regular)  | 1.4 (~20px) | Source badge labels, course tag pills, date text, toolbar icons, section note count             |
| Heading | 20px (typography.xl)     | 600 (semibold) | 1.2 (24px)  | Section headers, bottom sheet titles, editor title input, "AI Summary" / "Socratic Mode" titles |
| Display | 30px (typography['3xl']) | 700 (bold)     | 1.2 (36px)  | Empty state heading, quota exceeded warning                                                     |

Additional note-specific sizes:
| Element | Size | Weight | Notes |
|---------|------|--------|-------|
| Note title (in card) | 16px (base) | 600 (semibold) | Truncated to 1 line |
| Note title (in editor) | 24px ('2xl') | 700 (bold) | Max length 200 chars, placeholder "Note title..." |
| Streaming summary text | 16px (base) | 400 | lineHeight 24px explicitly set |
| Quick actions description | 14px (sm) | 400 | mutedForeground color |
| Toolbar icons | 14px (sm) | 400 | RichToolbar default size |

Font weight constraint: **2 weights only** — 400 (regular) and 600 (semibold). Display uses 700 as the single exception for the empty state heading and editor title.

│ │▌ 📄 PDF  ·  Mathematics  ·  2h │ │  ← Bottom row: source badge + course tag + date

**Layout specification:**

| Element        | Style                                                                                                                                                       |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Container      | flex:1, backgroundColor: #0f0f23, paddingTop: insets.top                                                                                                    |
| SearchBar      | marginHorizontal: 24px (lg), marginTop: 16px (md), backgroundColor: #1a1a2e, borderRadius: 12, padding: 12px (sm+md)                                        |

**Interaction:**

- Tap note card → navigate to NoteViewer

- Swipe right on note card → purple "Pin" action (80px width), toggles is_pinned
- Tap FAB → navigate to NoteEditor (new note)
- Search → filters by title with 300ms debounce
- Collapse/expand section header → toggles section body visibility

**Empty state:**

```

│                                     │

│            [＋ New Note]            │  ← Primary button
│                                     │
└─────────────────────────────────────┘
```

**Loading state:** ActivityIndicator with color #7c3aed, centered in container

**Responsive behavior:**

| Device                     | Notes list               | Cards per row                 | Section header |
